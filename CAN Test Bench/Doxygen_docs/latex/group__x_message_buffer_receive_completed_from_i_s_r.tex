The sb\+R\+E\+C\+E\+I\+V\+E\+\_\+\+C\+O\+M\+P\+L\+E\+T\+E\+D() macro is called from within the Free\+R\+T\+OS A\+P\+Is when data is read out of a message buffer or stream buffer. See the example implemented in Free\+R\+T\+O\+S/\+Demo/\+Minimal/\+Message\+Buffer\+A\+M\+P.\+c for additional information.
